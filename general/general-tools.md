## Tools

### [duckdns](https://www.duckdns.org/index.jsp)
> **Description:**
> `DuckDNS` is a free dynamic DNS (DDNS) service that allows you to assign a subdomain (e.g., `yourname.duckdns.org`) to your dynamic IP address. 
> It helps in situations where your IP address changes frequently, such as with most home internet connections, by automatically updating the DNS records to point to your current IP.  
>  
> DuckDNS is easy to set up and can be integrated into scripts or run on devices like Raspberry Pi to maintain a consistent domain name. 
> It supports several update methods, including API calls, cron jobs, and built-in integrations with platforms like Linux, Windows, and macOS.
>
